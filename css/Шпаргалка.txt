3️⃣ Мини-правила

Padding — внутрь блока, чтобы контент не прилипал к краям.

Margin — наружу блока, чтобы разделять элементы.

Вертикальные отступы чаще больше горизонтальных (особенно у секций и карточек).

Используем rem, чтобы всё масштабировалось с базовым шрифтом (html { font-size: 16px; }).

Для тонкой подгонки можно комбинировать calc():

padding: calc(1rem - 2px);

Правило “когда em, когда rem”

em — локальные пропорции относительно родителя (текст, padding, элементы в карточке)

rem — глобальные пропорции относительно root (html) (основные шрифты, отступы секций, размеры блоков)

-----------Button colors--------------------------------
Simple Rule Designers Use

For buttons:

State	Change
Normal	original colour
Hover	10–15% darker
Click	20–25% darker
Example:

State		HSL			Explanation
Default		hsl(25, 33%, 33%)	normal
Hover (-10%)	hsl(25, 33%, 30%)	slightly darker
Active (-20%)	hsl(25, 33%, 26%)	more pressed

So the calculation is simple:

33% - 3% ≈ 10% darker
33% - 7% ≈ 20% darker
